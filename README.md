# Go AI Translate

A command-line utility for translating text documents using the OpenRouter API.

### Example

```bash
export OPENROUTER_API_KEY="sk-AAAAA"
./go_ai_translate \
    --input docs/manual.txt \
    --output docs/manual_translated.txt \
    --api-key sk-AAAAA \
    --to ru \
    --verbose
```
